# Creative — Khanh's Content Production Workspace

> Video Editor | Vlogger | Content Creator | 20+ Years

## Folder Structure

| Folder | Dùng cho |
|---|---|
| `scripts/` | Script video hoàn chỉnh với timestamps và B-roll notes |
| `storyboards/` | Storyboard theo cảnh, visual planning |
| `shot-lists/` | Shot list pre-production theo video |
| `thumbnails/` | Thumbnail concept, mô tả visual, prompt AI |
| `series/` | Concept và outline cho series dài kỳ |
| `distribution/` | YouTube description, caption social, hashtag sets |
| `assets/` | Links tài nguyên, AI prompts, font/color refs |
| `strategy/` | Content calendar, analytics notes, brand direction |

## File Naming
- Script: `YYYY-MM-[topic]-script.md`
- Storyboard: `YYYY-MM-[topic]-storyboard.md`
- Shot list: `YYYY-MM-[topic]-shotlist.md`
- Series: `[series-name]-series-outline.md`

## Quick Start
Nói với Claude: *"Làm script cho video về..."* → Nhận được script đầy đủ + B-roll notes + thumbnail concept + title options.
