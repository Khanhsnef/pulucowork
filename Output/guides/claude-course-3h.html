📌 3 vùng chính của giao diện:

A

Bài 1.3 — Nghệ thuật viết Prompt
Kỹ năng quan trọng nhất khi dùng AI Quan trọng Thực hành
⏱ 15 phút
▼
🎯 Sau bài này bạn sẽ biết Viết prompt (câu lệnh) rõ ràng để Claude hiểu đúng và ra kết quả tốt ngay lần đầu.
📐 Template Prompt Chuẩn — 4 Thành Phần
VAI TRÒ
Bạn muốn Claude đóng vai gì? "Bạn là chuyên gia HR..."
NGỮ CẢNH
Tình huống cụ thể là gì? "Tôi cần viết email xin hoãn họp..."
YÊU CẦU
Cần làm gì chính xác? "Viết email lịch sự, súc tích..."
FORMAT
Muốn kết quả trông như thế nào? "Gồm: tiêu đề + nội dung, dưới 100 chữ"

So sánh: Prompt tệ vs Prompt tốt

❌ Prompt tệ

"viết email"

→ Claude không biết gửi cho ai, về chủ đề gì, tone thế nào → kết quả chung chung, vô dụng

✅ Prompt tốt

"Bạn là chuyên gia viết email công việc. Tôi cần hoãn cuộc họp ngày mai với đối tác vì lý do đột xuất. Viết email lịch sự, dưới 80 chữ. Format: Tiêu đề + Nội dung."

3 Kỹ thuật nâng cao

  • A
    Cho ví dụ mẫu: Paste một đoạn viết bạn thích → "Hãy viết theo phong cách tương tự"
  • B
    Giới hạn đầu ra: "Tối đa 200 chữ", "Liệt kê đúng 5 gạch đầu dòng", "Dưới dạng bảng"
  • C
    Hỏi lại nếu chưa ưng: "Làm lại nhưng tone thân thiện hơn", "Rút ngắn còn một nửa"
💡 Bài tập thực hành
Thử viết prompt theo template: yêu cầu Claude tóm tắt một cuốn sách bạn đã đọc hoặc một tài liệu PDF của bạn. Kiểm tra: kết quả có đúng format bạn muốn không?

Bài 2.2 — Tạo Project đầu tiên
Hướng dẫn từng bước thực hành Thực hành

▼
🎯 Sau bài này bạn sẽ có Một Project hoạt động với Instructions và Knowledge files — sẵn sàng dùng ngay.
  • 1
    Tạo Project mới: Sidebar trái → Nhìn thấy mục "Projects" → Nhấn "+ New project"
  • 2
    Đặt tên rõ ràng: Ví dụ: "Báo cáo Driver Ops 2026" hoặc "Nghiên cứu Đối thủ Q2" — tên càng rõ càng dễ tìm sau này.
  • 3
    Viết Instructions (quan trọng nhất!): Đây là "bản mô tả công việc" cho Claude trong project này. Ví dụ:
    Ví dụ Instructions cho Project Ahamove
    Bạn là trợ lý phân tích dữ liệu của team Driver Management,
    Ahamove. Nhiệm vụ chính: hỗ trợ phân tích KPI vận hành.
    // Quy tắc làm việc:
    - Luôn trả lời bằng tiếng Việt
    - Ưu tiên chỉ số: AR, FR, CPO, EPH, RPH
    - Trình bày dưới dạng bảng khi có nhiều số liệu
    - Kết thúc mỗi phân tích bằng mục "Khuyến nghị hành động"
  • 4
    Upload Knowledge files: Nhấn "Add content" → "Upload files" → Kéo thả file PDF, Excel, Word vào đây. Claude sẽ đọc và tham chiếu mỗi khi bạn hỏi.
  • 5
    Test: Nhấn "New conversation" trong Project → Hỏi thử một câu liên quan tài liệu bạn vừa upload → Kiểm tra Claude có dùng đúng tài liệu không.
💡 Mẹo viết Instructions hay
Instructions tốt = Claude không cần bạn giải thích lại mỗi lần. Hãy bao gồm: ngôn ngữ (tiếng Việt/Anh), vai trò (phân tích/viết lách/tóm tắt), format đầu ra (bảng/bullet/văn xuôi), và từ cấm (không dùng thuật ngữ X, không đề xuất Y...).

Web vs Desktop — So sánh trực quan

Tính năng🌐 Claude Web🖥️ Claude Desktop
Chat với Claude✅✅
Upload file từ web✅✅
Đọc file thẳng từ máy tính❌✅
Kết nối Gmail/Drive/Calendar❌✅ (qua MCP)
Mở Claude bằng phím tắt❌✅
Làm việc offline một phần❌✅

🔌 MCP là gì? — Giải thích đơn giản

Hãy nghĩ MCP (Model Context Protocol) như một "phiên dịch viên" — nó giúp Claude "nói chuyện" được với các phần mềm khác (Gmail, Google Drive, Notion...). Nhờ MCP, bạn có thể nói với Claude: "Tóm tắt 10 email chưa đọc của tôi" và Claude sẽ tự vào Gmail đọc rồi tóm tắt cho bạn.

🍎 Cài trên Mac

  • 1
    Vào claude.ai/download
  • 2
    Tải file .dmg (macOS)
  • 3

  • 1
    Vào claude.ai/download

  • 3

  • 4
    Claude tự mở sau khi cài xong. Nếu không, tìm trong Start Menu.
  • 5
    Đăng nhập tài khoản claude.ai của bạn

⌨️ Phím tắt hữu ích

Bài 3.3 — Kết nối MCP Server cơ bản
Cho Claude quyền đọc Gmail, Drive, Calendar Nâng cao
⏱ 10 phút
▼
🎯 Sau bài này bạn sẽ có Claude kết nối được với ít nhất một app (Google Drive hoặc Gmail).
  • 1
    Mở Claude Desktop → Nhấn vào menu "Claude" (góc trên trái, Mac) hoặc nút ⚙️ Settings (Windows)
  • 2
    Chọn tab "Integrations" hoặc "MCP" — bạn sẽ thấy danh sách các app có thể kết nối
  • 3
    Tìm Google Drive → Nhấn toggle để bật → Nhấn "Authorize" → Đăng nhập Google và cấp quyền
  • 4
    Tương tự với Gmail và Google Calendar nếu muốn
  • 5
    Test: Quay lại chat → Gõ: "Liệt kê 3 file gần nhất trên Google Drive của tôi"
⚠️ Lưu ý bảo mật
Chỉ kết nối với các app bạn hoàn toàn tin tưởng. Claude sẽ có quyền đọc (và đôi khi ghi) dữ liệu trong các app đó. Bạn có thể thu hồi quyền bất cứ lúc nào trong Settings → Integrations → Revoke.
✅ Kiểm tra thành công
Claude liệt kê được tên file thực từ Drive của bạn = kết nối thành công! Chúc mừng, bạn vừa mở khóa một sức mạnh hoàn toàn mới.

⚡ Cowork = Claude Code + Skills + Tools

🧰
Skills: Các "kỹ năng đặc biệt" được cài sẵn. Ví dụ: skill "xlsx" giúp Claude xử lý Excel chuyên nghiệp; skill "pptx" tạo PowerPoint đẹp ngay.
🔧
Tools: Kết nối sâu hơn với Gmail, Calendar, Drive, Lark... Claude tự động làm theo workflow phức tạp.
🤖
Automation: Đặt lịch tự động chạy task: "Mỗi sáng thứ 2 tóm tắt email tuần qua và gửi cho team".

Ví dụ workflow thực tế

  • 📊 Không dùng Cowork: Tải báo cáo → Mở Excel → Copy dữ liệu → Paste vào Claude Web → Yêu cầu phân tích → Copy kết quả → Mở Word → Paste → Format → 45 phút
  • ⚡ Dùng Cowork: Nói với Claude: "Phân tích báo cáo Driver_May2026.xlsx, tạo tóm tắt PDF và gửi email cho team@company.com" → Claude tự làm → 2 phút

→ Claude tự dùng skills /pdf + /pptx

Bài 5.2 — Cài đặt Claude Code từng bước
Hướng dẫn chi tiết, không bỏ bước nào Quan trọng Thực hành
⏱ 20 phút
▼
🎯 Sau bài này bạn sẽ có Claude Code chạy trên Terminal và sẵn sàng nhận lệnh đầu tiên.
1 Cài Node.js (nền tảng để chạy Claude Code)

🍎 Mac:

  1. Vào nodejs.org
  2. Nhấn nút xanh "Download LTS"
  3. Mở file .pkg tải về → Next → Install
  4. Nhập mật khẩu Mac nếu được hỏi

🪟 Windows:

  1. Vào nodejs.org
  2. Nhấn nút xanh "Download LTS"
  3. Chạy file .msi → Next → Next → Install
  4. Restart máy tính sau khi cài xong
Kiểm tra cài thành công
node --version # Phải thấy: v20.x.x hoặc cao hơn
2 Mở Terminal (cửa sổ dòng lệnh)

🍎 Mac:

Nhấn Cmd+Space → Gõ "Terminal" → Enter. Cửa sổ đen xuất hiện = thành công.

🪟 Windows:

Nhấn Windows+R → Gõ "cmd" → Enter. Hoặc tìm "PowerShell" trong Start Menu.

💡 Terminal trông đáng sợ nhưng không nguy hiểm
Terminal chỉ là một cách khác để ra lệnh cho máy tính. Thay vì click chuột, bạn gõ chữ. Claude Code sẽ hướng dẫn bạn cần gõ gì.
3 Cài Claude Code
Gõ lệnh này vào Terminal (copy và paste nguyên vẹn)
npm install -g @anthropic/claude-code
# Thấy nhiều dòng chữ chạy = bình thường, chờ 1-2 phút
# Thấy "added X packages" = cài thành công
Kiểm tra
claude --version # Phải thấy: 1.x.x
4 Lấy API Key (chìa khóa kết nối)
  • →
    Truy cập console.anthropic.com → Đăng nhập bằng tài khoản claude.ai
  • →
    Nhấn "API Keys" trong menu trái → Nhấn "+ Create Key"
  • →
    Đặt tên key (ví dụ: "Laptop cá nhân") → Copy key (bắt đầu bằng "sk-ant-...")
⚠️ Cực kỳ quan trọng — Bảo mật API Key
API Key này giống như mật khẩu ngân hàng. Không chia sẻ với ai, không paste vào chat, không đưa lên internet. Nếu lộ, vào console.anthropic.com để xóa ngay và tạo key mới.
5 Kết nối và chạy lần đầu tiên 🎉
Gõ lệnh này để khởi động Claude Code
claude
# Lần đầu: Claude sẽ hỏi API Key → paste key vào → Enter
# Thấy dòng chữ "Welcome to Claude Code" = thành công!
Test tin nhắn đầu tiên
"Xin chào! Hãy giới thiệu bản thân và những gì bạn có thể làm."
🎉 Chúc mừng!
Claude trả lời = bạn đã cài thành công Claude Code! Đây là cột mốc quan trọng nhất trong khóa học này.

📄 CLAUDE.md — "Hộ chiếu" của dự án

CLAUDE.md là file văn bản đặc biệt mà Claude Code đọc mỗi khi bạn mở project. Nó nói với Claude: "Đây là ai, project này là gì, làm việc theo cách nào." Có CLAUDE.md = không cần giải thích lại mỗi lần.

Tạo CLAUDE.md tự động (cách dễ nhất)
# Trong Claude Code, gõ lệnh:
/init
# Claude sẽ hỏi về dự án của bạn và tự tạo CLAUDE.md
