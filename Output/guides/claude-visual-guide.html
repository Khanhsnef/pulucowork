Quick Reference — Ghi vào đâu?
🌐 LUÔN → Global (~/.claude/)
"Hãy nhớ rằng…" · Cài skill /xlsx · Thêm MCP server · "Cho tất cả project"
🏢 Mặc định → Project (.claude/)
Sửa CLAUDE.md · Thêm permission · Tạo agent · Cài hook Stop · Không nói gì thêm
🔐 → Local (*.local.*)
"đừng commit" · "riêng tư" · "local only" · "chỉ máy này" · API keys
Nhận biết Claude viết vào đâu — xem đường dẫn file:
/Users/you/.claude/… → Global
Pulu-workspace/.claude/… → Project
*.local.md / *.local.json → Local
/Users/you/.claude/projects/… → Memory
💡 Không chắc? Hỏi thẳng: "Bạn vừa sửa file nào?"
📖 Giải thích thuật ngữ
Git / Commit
Git = hệ thống lưu lịch sử thay đổi file. Commit = lưu một checkpoint. "Không commit" = file chỉ tồn tại trên máy bạn, không chia sẻ.
Permission
Danh sách lệnh Claude được phép tự chạy mà không hỏi bạn. Ví dụ: allow npm run build.
Global vs Project
Global = áp dụng cho mọi dự án trên máy bạn. Project = chỉ áp dụng trong folder Pulu-workspace/ này thôi.
