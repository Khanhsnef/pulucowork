🌐 TOOL 1 — CLAUDE.AI PROJECTS
🌐
claude.ai / web
Claude.ai — Projects Mode
Làm việc theo Project với context cố định, knowledge base riêng
1
Tạo Project mới
Vào claude.ai → Sidebar trái → "New Project".
Đặt tên rõ ràng theo workstream: "Ahamove — Driver Tiering 2026", "Ahamove — Mega Sales Q3", "Personal — Smart Home"…
💡
Projects giúp Claude nhớ context xuyên suốt — không cần paste lại prompt mỗi cuộc trò chuyện.
2
Viết Project Instructions (System Prompt)
Trong Project → "Project Instructions" → paste system prompt định nghĩa role, output format, và constraints.

Template cho Ahamove Projects:
# Dán vào ô Project Instructions
Bạn là AI hỗ trợ Lê Phương Khanh, Driver Management Leader tại Ahamove.
Scope: xe máy Instant (Giao ngay 1H, Siêu tốc, Ghép đơn, 4H).
METRICS THEO DÕI:
- AR (Acceptance Rate), FR (Fulfillment Rate)
- CPO, EPH, RPH, GDR, CTR
OUTPUT RULES:
- Trả lời tiếng Việt, thuật ngữ giữ tiếng Anh
- Bảng > bullet > văn xuôi
- Pyramid Principle: Kết luận → Lý do → Data
- Không bịa số, không lời khuyên chung chung
3

📊

🗄️

.sql · .md
🎯

🏎️

.pdf · .docx
📑

⚠️
Giới hạn 200K tokens (~500 trang). Ưu tiên upload tài liệu evergreen (rules, schema, policy) thay vì data real-time.

🖥️ TOOL 2 — CLAUDE DESKTOP COWORK
🖥️
claude desktop / mac app
Claude Desktop — Cowork Mode
App native, tích hợp MCP tools, đọc file local, chạy skills nâng cao
1
Cài đặt Claude Desktop
Download tại claude.ai/download → Chọn macOS → Cài đặt bình thường.
Đăng nhập cùng tài khoản Anthropic (Max Plan).
📌
Claude Desktop ≠ claude.ai web. Desktop có khả năng kết nối MCP servers và đọc file local — web không có.
2
Cấu trúc Cowork Folder
Toàn bộ output, context, và config được tổ chức trong ~/Desktop/Cowork/:
# Cấu trúc thư mục Cowork
~/Desktop/Cowork/
├── CLAUDE.md # ← System prompt tổng (role, rules, brand)
├── .claude/ # ← Config: settings.json, hooks, skills
│ ├── settings.json # Permissions, MCP servers, hooks
│ └── memory/ # Auto-memory: user profile, feedback
└── output/ # Tất cả file output theo cấu trúc
├── Ahamove/ # 01~07 folders
├── guides/ # File này đang ở đây!
└── Personal/
3
MCP Servers đang kết nối
Các MCP servers cho phép Claude Desktop truy cập tools bên ngoài:
📧 Gmail MCP
Đọc, soạn, label email khanhlp@ahamove.com
📅 Google Calendar
Tạo, xem, cập nhật sự kiện; gợi ý thời gian
📁 Google Drive
Đọc, tìm kiếm, copy file trên Drive
📓 NotebookLM MCP
Tạo notebook, query, studio audio/video
📊 Aira Analytics
Query BigQuery (read-only) — AR, FR, CPO data
📋 Notion / Atlassian
Kết nối khi cần — auth flow riêng
4
Skills hỗ trợ trong Cowork
Gọi skill bằng cách gõ /tên-skill trong chat:
📑
/pptx · /pdf · /docx
Tạo presentation, PDF, Word document chuyên nghiệp
📊
/xlsx · /xlsx-a
Financial model, spreadsheet, 3-statement model (IB standard)
🌐
/html · /md
Report HTML McKinsey-style, markdown document
🎯
/okr · /plan
Build OKR quarterly, lập kế hoạch đa bước phức tạp
🏠
/home-design
Thiết kế nội thất, phong thủy, layout phòng
⏰
/schedule · /loop
Lên lịch task tự động, chạy lặp theo interval
5
Tips làm việc hiệu quả với Cowork
✓
Luôn mở Claude Desktop từ folder ~/Desktop/Cowork để Claude nhận CLAUDE.md đúng
✓
Dùng "Extended Thinking" (nút ⚡ trong chat) cho phân tích phức tạp cần suy luận sâu
✓
File output tạo ra → Lưu vào đúng folder output/Ahamove/XX.FOLDER/
!
Khi Claude xin quyền tool (Gmail, Drive) → approve từng lần hoặc dùng /fewer-permission-prompts
⚡ TOOL 3 — CLAUDE CODE / ANTIGRAVITY
⚡
claude code / terminal / antigravity mac
Claude Code — Terminal Agent
CLI agent chạy trong terminal, tự đọc/ghi file, chạy shell, git, deploy
1
Cài đặt Claude Code trên Antigravity
Chạy trong Terminal (zsh):
# Bước 1: Cài Node.js nếu chưa có (via Homebrew)
brew install node
# Bước 2: Cài Claude Code global
npm install -g @anthropic-ai/claude-code
# Bước 3: Login (mở browser, authenticate)
claude
# → Lần đầu: paste Anthropic API key hoặc OAuth
# Kiểm tra version
claude --version
✅
Trên Antigravity đã cài sẵn — mở terminal, cd ~/Desktop/Cowork rồi gõ claude là chạy ngay.
2
CLAUDE.md — "Hồ sơ công việc" của dự án
File CLAUDE.md trong thư mục làm việc là system prompt tự động — Claude Code đọc mỗi khi khởi động. Đây là nguồn context quan trọng nhất:
# Cấu trúc CLAUDE.md hiện tại (~/Desktop/Cowork/CLAUDE.md)
## 🏢 Company & Role Context
- Khanh: Driver Management Leader, Ahamove
- Scope: Bike Instant (Giao ngay, Siêu tốc, Ghép đơn)
## Brand
- Colors: #0E4174 (Blue), #FF7F32 (Orange)
- Fonts: Lexend
## Output Rules & File Naming
- output/Ahamove/01~07 folder structure
- Naming: YYYY-MM-topic.ext (lowercase, hyphens)
💡
CLAUDE.md được load tự động — không cần paste lại mỗi session. Nếu cần cập nhật rules, chỉnh trực tiếp file này.
3
Cách khởi động và giao việc
INTERACTIVE MODE (thường dùng)
# Mở Claude Code trong Cowork
cd ~/Desktop/Cowork
claude
# → Giao tiếp bằng chat trong terminal
# → Claude tự đọc CLAUDE.md
# → Dùng skill: /html /xlsx /pdf ...
ONE-SHOT MODE (task nhanh)
# Chạy task rồi thoát luôn
claude "Tạo file HTML báo cáo AR tuần này,
lưu vào output/Ahamove/04.OPS_METRICS/"
# Với file input
claude "Đọc data.csv, phân tích CPO"

6
settings.json — Cấu hình permission & hooks
File ~/.claude/settings.json kiểm soát những gì Claude Code được phép làm tự động:
{
"permissions": {
"allow": [
"Read", "Edit", "Write", "Bash(git *)",
"Bash(ls *)", "Bash(find *)", "Bash(grep *)"
]
},
"hooks": {
"PostToolUse": // chạy script sau mỗi tool call
"Stop": // notification khi Claude xong việc
}
}
⚙️
Dùng /fewer-permission-prompts để Claude tự scan history và tự động thêm các tool hay dùng vào allowlist.
7
Memory System — Claude nhớ qua các session
Claude Code có hệ thống memory tự động lưu vào ~/.claude/projects/…/memory/:
👤
user_profile.md
Role, background, cách Khanh làm việc, metrics theo dõi
📝
user_rules.md
Rules: brief trước, chờ confirm, SQL first, Pyramid Principle
🎨
user_voice.md
Format output: bảng > bullet > văn xuôi, từ vựng chuyên ngành
💬
feedback_*.md
Ghi lại corrections + confirmations — tránh lặp lại lỗi cũ
💾
Khi muốn Claude nhớ điều gì: gõ "hãy nhớ…" — Claude tự lưu vào memory file đúng loại. Memory này persist qua tất cả sessions.
